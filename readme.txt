Lock your content by asking users to like your page on Facebook first.

== Description == 

Locks your content and presents the user with a Facebook Like button. Liking the page then reveals the content.

* Enable post or page wide locks
* Lock specific blocks of content using the shortcode
* Change the look of your LikeLock using the shortcode wizard
* Increases your Facebook likes


== Installation ==

1. Upload the 'pluginbear_likelock' folder to the '/wp-content/plugins' directory 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to PluginBear > LikeLock and add your Facebook App ID
4. That is all!

== Frequently Asked Questions ==

= Why do I need a Facebook App ID? =

The plugin uses the Facebook API to determine whether a user as liked your page or not. Without this, the plugin wouldn't work.
